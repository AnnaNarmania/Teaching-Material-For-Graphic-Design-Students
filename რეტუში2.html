01 · ძირითადი ინსტრუმენტები

Content-Aware · Red Eye · Color Replacement

სამი ძირითადი ინსტრუმენტი, რომელიც ფოტოს ხარვეზებს — "წითელ თვალს", არასასურველ ობიექტებს და არასწორ ფერებს — პროფესიულად ასწორებს.

✦

Content-Aware Fill / Move / Scale

Photoshop-ის ყველაზე "ჯადოსნური" ფუნქცია — ირჩევ სასურველ ობიექტს და პროგრამა ავტომატურად ავსებს ახლო ტექსტურით.

Content-Aware Fill — Edit → Fill → Content-Aware. გამოყენება: ობიექტის წაშლა, ლანდშაფტის გამრავლება.
Content-Aware Move Tool (J) — ობიექტს გადაათრევ ახალ ადგილას, ძველ ადგილს ავტომატურად ავსებს.
Content-Aware Scale — Edit → Content-Aware Scale: სურათს ზომაში ცვლი, ობიექტები "ჭყლეტა" — ადამიანი ან ძირითადი ობიექტი ინახება.

გამოიყენება: ფოტოდან ადამიანის, დაგრეხილი ხაზების, მავთულის, ჩრდილის წაშლა.
J (Move) · Shift+F5 (Fill) Edit → Fill → Content-Aware
👁

Red Eye Tool

ფლეშ-ფოტოგრაფიის "წითელი თვალის" სინდრომის ერთ-ორ დაჭერით გამოსწორება.

Pupil Size — გუგის ზომა (0–100%). მეტი = პროგრამა უფრო დიდ წრეს ასწორებს.
Darken Amount — დამუქება: რამდენად ბნელი გახდება ასწორებული გუგა. 50% — ნორმალური; 100% — სრულად შავი.

გამოყენება: ირჩევ Red Eye Tool-ს (J გრძელი დაჭერა), ერთხელ დააჭერ წითელ გუგაზე — მზადაა.

Pet Eye: ცხოველებს ხშირად მწვანე ან თეთრი ანარეკლი აქვთ — Red Eye Tool Pet Eye mode-ით მუშაობს.
J (გრძელი დაჭერა → Red Eye Tool) Toolbar → J → Red Eye Tool
🖌

Color Replacement Tool

ფუნჯის მსგავსად — სვამ ახალ ფერს, პროგრამა კი ავტომატურად ინახავს ტექსტურასა და ჩრდილს. ანუ ფერი იცვლება, ტონები და ნახატი — არა.

Mode: Hue (ფერი), Saturation (გაჯერება), Color (Hue+Sat), Luminosity (სინათლე).
Sampling: Once — პირველ დაჭერაზე ნიმუშს იღებს (იდეალური ერთფეროვანი ზედაპირისთვის); Continuous — მუდმივად; Background Swatch — მხოლოდ ფონის ფერს ცვლის.
Limits: Contiguous — ახლო პიქსელები; Find Edges — კიდეებს ინახავს.
Tolerance — რამდენი ახლო ფერი ჩაეთვლება (0–100%).

vs Replace Color (Adjustment): Color Replacement Tool ფუნჯია — ხელით ხატავ; Replace Color — მთელ სურათზე ერთდროულად.
B → Color Replacement Tool Toolbar → B (გრძელი) → Color Replacement

Content-Aware — პრაქტიკული ნაბიჯები

1. Lasso Tool-ით (L) შემოხაზე სასურველი ობიექტი — 10–15px მოშვებით.
2. Edit → Fill → Contents: Content-Aware → OK.
3. Select → Deselect (Ctrl+D). შედეგი — 1–3 წამში.
რჩევა: რთულ ტექსტურებზე (ბალახი, ქვა) — Healing Brush Tool-ით დაამატებ კორექცია. სუფთა ცა ან ერთფეროვანი ფონი — შედეგი თითქმის სრულყოფილია.

02 · პორტრეტის რეტუში

პროფესიული პორტრეტი

პორტრეტის რეტუში — ნაბიჯ-ნაბიჯ. კანი, თვალები, ტონი და სტრუქტურა სრული კონტროლქვეშ.

01
Spot Healing Brush — წვრილი ხარვეზები
კანის წვრილი ხარვეზები — აკნე, ნაწიბური, ლაქა — ერთ დაჭერით. ირჩევ Spot Healing Brush (J), ფუნჯის ზომა ხარვეზზე ოდნავ დიდი, ერთხელ დააჭირე. Content-Aware ავტომატურად ავსებს ახლო ტექსტურით.
Type: Content-Aware → ყველაზე კარგი შედეგი.
J → Spot Healing Brush
02
Healing Brush — ტექსტურის შენარჩუნება
Healing Brush (J) — Alt+Click ნიმუშის ასაღებად, შემდეგ — ხარვეზზე. ტექსტურა ინახება, ფერი და ტონი — ახლო პიქსელებთან იზავება. იდეალური: დიდი ლაქები, ნაოჭები, ჩრდილები.
J → Healing Brush · Alt+Click = ნიმუში
03
Frequency Separation — კანის გლუვება
პროფესიული ტექნიკა: სურათს ორ ფენად ყოფ — Low Frequency (ფერი და ტონი) და High Frequency (ტექსტურა). Low-ზე Gaussian Blur + Clone/Heal-ით გლუვებ ფერს, High — ტექსტურა ხელუხლებელია.
ნაბიჯები: Duplicate ×2 → High: Filter → Other → High Pass (2–4px) → Blend Mode: Linear Light → Low: Filter → Gaussian Blur (2–4px).
Layer → Duplicate × 2 · High Pass + Gaussian Blur
04
Dodge & Burn — მოცულობა და ჩრდილი
Dodge Tool (O) — ანათებს (ყბა, ცხვირის ხაზი, თვალქვეშა); Burn Tool (O) — ამუქებს (ყბის კიდე, ჩრდილი). Range: Midtones — კანისთვის; Highlights — გამბრწყინვალო წერტილებისთვის; Exposure: 3–8% — ნატიფი შედეგი.
ახალ ნეიტრალურ ფენაზე (Blend: Overlay/Soft Light) — არადესტრუქციული.
O → Dodge / Burn · Exposure: 3–8%
05
Liquify — კონტურის კორექცია
Filter → Liquify (Shift+Ctrl+X). Face-Aware Liquify — ავტომატურად პოულობს სახის ელემენტებს: თვალების ზომა, ცხვირი, ტუჩები, სახის ფორმა. Forward Warp Tool (W) — ხელით ათრევ. Freeze Mask — ფიქსირებ ზონებს. Exposure: ძალიან ნელ-ნელა, ბუნებრივობის შენარჩუნებით.
Filter → Liquify (Shift+Ctrl+X)
06
Color Grading — ფინალური ტონი
Adjustment Layer-ები: Curves — ტონი; Hue/Saturation — ფერი; Color Balance — ჩრდილი/Midtones/Highlights. Selective Color-ით კანის ტონი: Reds/Yellows → Cyan/Magenta-ს რეგულაცია. Camera Raw Filter-ი — Clarity, Texture, Dehaze სრული კონტროლისთვის.
Layer → New Adjustment Layer → Curves / Hue-Sat

🔑 მთავარი პრინციპი — არადესტრუქციული სამუშაო

ყველა ეტაპი — ცალკე ფენაზე. Original ფენა ყოველთვის ხელუხლებელი. Smart Object-ებად გადაქცევა (Right Click → Convert to Smart Object) Filter-ების შეცვლის საშუალებას იძლევა. Adjustment Layers — Mask-ებით. Dodge & Burn — ახალ ცარიელ ფენაზე (Overlay blend).

შედეგი: ნებისმიერ ეტაპზე შეგიძლია დაბრუნება და Opacity-ის შეცვლა — ბუნებრივობა ყოველთვის კონტროლქვეშ.

სრული შედარება

🔴 Red Eye vs Color Replacement

  • Red Eye — ავტო, ერთი კლიკი, გუგა
  • Color Replacement — ხელი, ნებისმიერი ფერი
  • Red Eye = სწრაფი; CR = მოქნილი

✦ Content-Aware Fill vs Move

  • Fill — ობიექტი ქრება, ავსება
  • Move — ობიექტი გადაადგილება + ავსება
  • Scale — ზომა, ობიექტი ინახება

🖌 Spot Healing vs Healing Brush

  • Spot Healing — ნიმუში არ სჭირდება
  • Healing Brush — Alt+Click ნიმუში
  • Healing = კონტროლი; Spot = სიჩქარე
