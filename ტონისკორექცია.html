01 · ტონალური ინსტრუმენტები

Levels · Curves · Shadows/Highlights

ეს სამი ინსტრუმენტი ფოტოს ტონალური სტრუქტურის ბირთვია — სინათლის, ჩრდილისა და კონტრასტის სრული კონტროლი. Image → Adjustments

📊

Levels

ჰისტოგრამაზე დაფუძნებული ინსტრუმენტი, რომელიც სამ წერტილს — Shadows, Midtones, Highlights — ცვლის. Input-ის ლიანდაგები შეკუმშავს ტონალურ დიაპაზონს, Output კი ამცირებს კონტრასტს. მარჯვნივ Input Shadows-ის გადაწევა — სურათი ბნელდება; Highlights-ის მარცხნივ — ნათელდება.

გამოიყენება: ექსპოზიციის კორექცია, კონტრასტის ბაზური კონტროლი.
Ctrl + L / Cmd + L Image → Adjustments → Levels
〜

Curves

Curves — Levels-ის გამდიდრებული ვერსია. დიაგონალური ხაზი ნებისმიერ წერტილში მოხრით შეიძლება ცვლიდე კონკრეტულ ტონებს. S-ფორმის მრუდი — კლასიკური კონტრასტი. ზედა ნახევრის ამოწევა — Highlights ნათლდება; ქვედა ჩამოწევა — Shadows ბნელდება. ასევე მუშაობს ცალ-ცალკე R, G, B არხებზე ფერის კორექციისთვის.

vs Levels: Curves გაცილებით მეტ სიზუსტეს იძლევა — ერთდროულად მრავალი ტონის კონტროლი.
Ctrl + M / Cmd + M Image → Adjustments → Curves
◑

Shadows / Highlights

სპეციფიკური ინსტრუმენტი, რომელიც Shadows-სა და Highlights-ს ცალ-ცალკე, დამოუკიდებლად ცვლის — ჩრდილი ნათელდება პირდაპირ, ჰაილაიტი ბნელდება. Advanced Options-ში ასევე Midtone Contrast, Color Correction და Radius (ლოკალური სიგანე).

იდეალური: გადამუქებული ჩრდილის გახსნა ან გადანათებული ცის ჩამოწევა.
Image → Adjustments → Shadows/Highlights
Levels · ჰისტოგრამა

ჰისტოგრამა — როგორ წავიკითხოთ?

ჰისტოგრამა გვიჩვენებს პიქსელების განაწილებას ბნელიდან (მარცხნივ) ნათლამდე (მარჯვნივ). მარცხნივ დაჭიმული პიკი — სურათი ბნელია; მარჯვნივ — ნათელი. შუაში — ნორმალური ექსპოზიცია. Input-ის სამი ლიანდაგი (▲ ▲ ▲) — მარცხენა = Shadows, შუა = Gamma/Midtones, მარჯვენა = Highlights.

02 · სპეციალური ეფექტები

Variations · Replace Color · Posterize · Threshold

ეს ინსტრუმენტები ფოტოს ვიზუალური ეფექტების, ფერის ჩანაცვლებისა და სტილიზაციისთვის გამოიყენება.

🔀
Variations
ვიზუალური ინტერფეისი ფერის კორექციისთვის — thumbnail-ების სახით.
Color
გვიჩვენებს ფოტოს ვარიაციებს: მეტი წითელი, მეტი მწვანე, ნათელი, ბნელი და ა.შ. ვარიანტების დაჭერა ინტუიციურად ცვლის ფერს.
Shadow/Midtones/Highlights — სხვადასხვა ზონაზე გამოყენება.
Coarse/Fine — ცვლილების სიძლიერე.
შენიშვნა: Variations არ მუშაობს 32-bit სურათებზე. Image → Adjustments → Variations
🎨
Replace Color
კონკრეტული ფერის არეს Hue, Saturation, Lightness-ით ჩანაცვლება.
Color
Eyedropper-ით ირჩევ სამიზნე ფერს, შემდეგ Fuzziness ფლობს, კარგად ისწავლე (0–200) — რამდენი ახლო ფერი შედის.
Hue — ფერი იცვლება; Saturation — გაჯერება; Lightness — სინათლე.
Preview checkbox გვაჩვენებს ცვლილებებს პირდაპირ. Image → Adjustments → Replace Color
🟫
Posterize
ფერების რაოდენობის შემცირება — ბეჭდვის პლაკატის ეფექტი.
Effect
Levels პარამეტრი (2–255) — რამდენი ტონი დარჩება.
2 = ძალიან მკვეთრი, პლაკატური; 4–8 = მხატვრული; 255 = ცვლილება თითქმის არ ჩანს.
გამოიყენება: გრაფიკული ხელოვნება, კომიქსის სტილი, printmaking ეფექტები. Image → Adjustments → Posterize
◼◻
Threshold
ყველა ფერი — მხოლოდ შავი ან მხოლოდ თეთრი. ზღვარი შენ განსაზღვრავ.
Effect
Threshold Level (1–255) — ამ მნიშვნელობაზე ნათელი პიქსელი = თეთრი, ბნელი = შავი.
დაბალი მნიშვნელობა (50-ის ქვემოთ) — სურათი უმეტესად შავია; მაღალი (200+) — უმეტესად თეთრი.
გამოიყენება: სილუეტის, ლოგოს, სტენციალის შექმნა. ასევე ჰისტოგრამის ანალიზისთვის. Image → Adjustments → Threshold
🌈
Gradient Map
ტონებს ანაცვლებს გრადიენტის ფერებით — ბნელი ტონი → გრადიენტის მარცხენა ბოლო.
Creative
სურათის სინათლის ინტენსივობა გრადიენტის პოზიციად გადაიქცევა: ყველაზე ბნელი = გრადიენტის მარცხენა ფერი; ყველაზე ნათელი = მარჯვენა ფერი.
შავ-თეთრი გრადიენტი = ფოტო B&W; ოქრო-ლურჯი = დუოტონი; ცისარტყელა = ფსიქოდელური.
Dither — ფოტო-ნოიზი (smoother gradients). Reverse — გრადიენტი შებრუნება. Image → Adjustments → Gradient Map
🎯
Selective Color
კონკრეტული ფერების ჯგუფში CMYK-ის პროცენტების სელექტიური შეცვლა.
Pro
ირჩევ ფერთა ჯგუფს: Reds, Yellows, Greens, Cyans, Blues, Magentas, Whites, Neutrals, Blacks.
შემდეგ ცვლი Cyan, Magenta, Yellow, Black-ის %. მხოლოდ ის ფერები იცვლება, სხვა — არა.
Relative — ნომინალური ცვლა (%-ის მიმართ); Absolute — ზუსტი %.
გამოიყენება: ცის ლურჯობის გაძლიერება, კანის ტონის კორექცია, ბალახის მოყვითლება. Image → Adjustments → Selective Color

03 · შედარება

ინსტრუმენტების კლასიფიკაცია

ინსტრუმენტები პრაქტიკული გამოყენების მიხედვით.

ტონალური კორექცია

ტონი და კონტრასტი

სინათლის, ჩრდილისა და კონტრასტის სამართავი ინსტრუმენტები.

  • Levels ბაზური ტონალური კონტროლი
  • Curves ზუსტი, მრავალწერტილიანი
  • Shadows/Highlights ლოკალური ზონის კორექცია
ფერის კორექცია

ფერი და ჰუე

ფერის ჩანაცვლება, არჩევა, გაჯერება.

  • Variations ვიზუალური ფერის კორექცია
  • Replace Color ფერის სელექტიური ჩანაცვლება
  • Selective Color CMYK-ით ჯგუფური კონტროლი
სტილიზაცია

კრეატიული ეფექტები

ფოტოს ხელოვნებად გარდაქმნა.

  • Posterize პლაკატური / კომიქს სტილი
  • Threshold შავ-თეთრი სილუეტი
  • Gradient Map ტონალური ფერის მაპინგი
სიძლიერის შედარება

Levels vs Curves

ყველაზე ხშირი კითხვა — რომელი გამოვიყენო?

  • Levels სწრაფი, ბაზური, 3 წერტილი
  • Curves ზუსტი, მოქნილი, პროფესიული
  • Curves = Levels + გაცილებით მეტი
სრული შედარება

⚡ სწრაფი გამოყენება

  • Levels — სწრაფი ექსპოზიციის კორექცია
  • Variations — ინტუიციური ფერი (ახალბედა)
  • Threshold — სილუეტი ერთი ნაბიჯით
  • Posterize — მყისიერი ეფექტი

🎯 პროფესიული კონტროლი

  • Curves — ტონის ზუსტი მართვა
  • Selective Color — CMYK-ის ოსტატური გამოყენება
  • Replace Color — სელექტიური ჩანაცვლება
  • Shadows/Highlights — ლოკალური ოპტიმიზაცია

🖼 ფოტო-კორექცია

  • Levels + Curves — ტონის სტრუქტურა
  • Shadows/Highlights — გადა-ექსპოზირებული ზონები
  • Selective Color — ბუნებრივი ფერი (ლანდშაფტი)

🎨 კრეატიული სტილი

  • Gradient Map — დუოტონი / ვინტაჟი
  • Posterize — გრაფიკული ხელოვნება
  • Threshold — სტენციალი, ლოგო
