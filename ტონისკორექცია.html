01 · ტონალური ინსტრუმენტები

Levels · Curves · Shadows/Highlights

ეს სამი ინსტრუმენტი ფოტოს ტონალური სტრუქტურის ბირთვია — სინათლის, ჩრდილისა და კონტრასტის სრული კონტროლი. Image → Adjustments

📊

Levels

ჰისტოგრამაზე დაფუძნებული ინსტრუმენტი, რომელიც სამ წერტილს — Shadows, Midtones, Highlights — ცვლის. Input-ის ლიანდაგები შეკუმშავს ტონალურ დიაპაზონს, Output კი ამცირებს კონტრასტს.

გამოიყენება: ექსპოზიციის კორექცია, კონტრასტის ბაზური კონტროლი.
Ctrl + L / Cmd + L Image → Adjustments → Levels
〜

Curves

Curves — Levels-ის გამდიდრებული ვერსია. დიაგონალური ხაზი ნებისმიერ წერტილში მოხრით შეიძლება ცვლიდე კონკრეტულ ტონებს. S-ფორმის მრუდი — კლასიკური კონტრასტი. ასევე მუშაობს ცალ-ცალკე R, G, B არხებზე.

vs Levels: Curves გაცილებით მეტ სიზუსტეს იძლევა — ერთდროულად მრავალი ტონის კონტროლი.
Ctrl + M / Cmd + M Image → Adjustments → Curves
◑

Shadows / Highlights

სპეციფიკური ინსტრუმენტი, რომელიც Shadows-სა და Highlights-ს ცალ-ცალკე, დამოუკიდებლად ცვლის. Advanced Options-ში — Midtone Contrast, Color Correction და Radius.

იდეალური: გადამუქებული ჩრდილის გახსნა ან გადანათებული ცის ჩამოწევა.
Image → Adjustments → Shadows/Highlights
Levels · ჰისტოგრამა

ჰისტოგრამა — როგორ წავიკითხოთ?

ჰისტოგრამა გვიჩვენებს პიქსელების განაწილებას ბნელიდან (მარცხნივ) ნათლამდე (მარჯვნივ). მარცხნივ დაჭიმული პიკი — სურათი ბნელია; მარჯვნივ — ნათელი. Input-ის სამი ლიანდაგი (▲ ▲ ▲) — მარცხენა = Shadows, შუა = Gamma/Midtones, მარჯვენა = Highlights.

02 · სპეციალური ეფექტები

Variations · Replace Color · Posterize · Threshold

ეს ინსტრუმენტები ფოტოს ვიზუალური ეფექტების, ფერის ჩანაცვლებისა და სტილიზაციისთვის გამოიყენება.

🔀
Variations
ვიზუალური ინტერფეისი ფერის კორექციისთვის — thumbnail-ების სახით.
Color
გვიჩვენებს ფოტოს ვარიაციებს: მეტი წითელი, მეტი მწვანე, ნათელი, ბნელი და ა.შ. Shadow/Midtones/Highlights — სხვადასხვა ზონაზე გამოყენება. Coarse/Fine — ცვლილების სიძლიერე. შენიშვნა: Variations არ მუშაობს 32-bit სურათებზე.Image → Adjustments → Variations
🎨
Replace Color
კონკრეტული ფერის არეს Hue, Saturation, Lightness-ით ჩანაცვლება.
Color
Eyedropper-ით ირჩევ სამიზნე ფერს, შემდეგ Fuzziness (0–200) — რამდენი ახლო ფერი შედის. Hue — ფერი; Saturation — გაჯერება; Lightness — სინათლე. Preview checkbox გვაჩვენებს ცვლილებებს პირდაპირ.Image → Adjustments → Replace Color
🟫
Posterize
ფერების რაოდენობის შემცირება — ბეჭდვის პლაკატის ეფექტი.
Effect
Levels პარამეტრი (2–255) — რამდენი ტონი დარჩება. 2 = ძალიან მკვეთრი; 4–8 = მხატვრული; 255 = თითქმის ცვლილება არ ჩანს. გამოიყენება: გრაფიკული ხელოვნება, კომიქსის სტილი.Image → Adjustments → Posterize
◼◻
Threshold
ყველა ფერი — მხოლოდ შავი ან მხოლოდ თეთრი. ზღვარი შენ განსაზღვრავ.
Effect
Threshold Level (1–255) — ამ მნიშვნელობაზე ნათელი პიქსელი = თეთრი, ბნელი = შავი. გამოიყენება: სილუეტის, ლოგოს, სტენციალის შექმნა.Image → Adjustments → Threshold
🌈
Gradient Map
ტონებს ანაცვლებს გრადიენტის ფერებით — ბნელი ტონი → გრადიენტის მარცხენა ბოლო.
Creative
სურათის სინათლის ინტენსივობა გრადიენტის პოზიციად გადაიქცევა. შავ-თეთრი გრადიენტი = B&W; ოქრო-ლურჯი = დუოტონი. Dither — smoother gradients. Reverse — შებრუნება.Image → Adjustments → Gradient Map
🎯
Selective Color
კონკრეტული ფერების ჯგუფში CMYK-ის პროცენტების სელექტიური შეცვლა.
Pro
ირჩევ ჯგუფს: Reds, Yellows, Greens, Cyans, Blues, Magentas, Whites, Neutrals, Blacks. შემდეგ ცვლი C, M, Y, K %-ს. Relative — ნომინალური; Absolute — ზუსტი %.Image → Adjustments → Selective Color

03 · სახელოსნო ინსტრუმენტები

Dodge · Burn · Sponge

სამი ინსტრუმენტი, რომლებიც პირდაპირ კანვასზე სურათის კონკრეტულ ზონებს ანათებს, ამუქებს ან გაჯერებას/გაუფერულებას ახდენს. ყველა მათგანი Toolbox-შია — კლავიში O

☀️
Dodge Tool
O · Dodge

ფოტოს კონკრეტულ ადგილს ანათებს. სახელი მომდინარეობს ფოტოლაბორატორიული ტექნიკიდან — ბეჭდვის დროს შუქის გვერდის ავლა (dodging).

Range — სად მუშაობს
Shadows Midtones Highlights
Exposure — სიძლიერე

1%–100%. დაბალი % (10–20%) — ნაზი, შეუმჩნეველი გაანათება; მაღალი % — ძლიერი, სწრაფი.

Protect Tones ✓

ჩართვისას — ფერი ნაკლებად იცვლება, მხოლოდ სინათლე. გამორთვისას — გადანათება ფერსაც ამახინჯებს.

როდის გამოვიყენოთ
  • თვალის, კბილის, სახის გასაბრწყინებლად პორტრეტში
  • ცის ღრუბლების გასაანათებლად
  • ობიექტის კონტურის გასაოქროებლად
  • ჩრდილში ჩახრჩობილი დეტალის გამოსაყვანად
🔥
Burn Tool
O · Burn

ფოტოს კონკრეტულ ადგილს ამუქებს — Dodge-ის საპირისპირო. ფოტოლაბორატორიაში "burning" ნიშნავდა ბეჭდვის დროს ქაღალდს მეტი შუქის მიცემას.

Range — სად მუშაობს
Shadows Midtones Highlights
Exposure — სიძლიერე

დაბალი % — ნაზი ვინიეტი, depth; მაღალი % — სწრაფი, ძლიერი დამუქება.

Protect Tones ✓

Dodge-ის მსგავსად — ფერის შენარჩუნება. ჩართულია ყოველთვის სასარგებლო.

როდის გამოვიყენოთ
  • ვინიეტის (კუთხის დამუქება) ხელით შექმნა
  • პორტრეტში ჩრდილის გაღრმავება — სახის ძვლების ხაზი
  • ცის გადანათებული ზონის ჩამოწევა
  • თმის, ჟანგის, ქვის ფაქტურის გამოკვეთა
💧
Sponge Tool
O · Sponge

ფოტოს კონკრეტულ ზონაში ფერის გაჯერებას (Saturate) ან გაუფერულებას (Desaturate) ახდენს. Dodge/Burn-ისგან განსხვავებით — სინათლეს არ ეხება, მხოლოდ ფერს.

Mode — რეჟიმი
Saturate Desaturate
Flow — სიძლიერე

Flow % — 1–100. დაბალი = ნელი, კუმულაციური ეფექტი; მაღალი = სწრაფი ცვლილება.

Vibrance ✓

ჩართვისას — Saturate რეჟიმი ფრთხილად მოქმედებს: უკვე გაჯერებული ფერი ნაკლებად ჭარბდება (ანალოგია Vibrance-ს).

როდის გამოვიყენოთ
  • თვალის ფერის ლოკალური გაძლიერება
  • ფონის სელექტიური გაუფერულება — სუბიექტი გამოიყოფა
  • ტუჩების, ყვავილის ფერის გაცხოველება
  • B&W გარდამავალი ეფექტი — ნაწილი ფერადი, ნაწილი პრუნი
სამივეს შედარება
ინსტრუმენტი მოქმედება Range / Mode სიძლიერის პარამ. ანალოგი

💡

Dodge vs Burn — Range-ის მნიშვნელობა

Dodge + Highlights — ნათელ ზონებს კიდევ უფრო ანათებს (ღრუბელი, კბილი). Dodge + Shadows — ჩრდილს "ხსნის" პირდაპირ, ნათელ ზონებს თავს ანებებს.
Burn + Highlights — გადანათებული ზონების ჩამოწევა. Burn + Shadows — ჩრდილების გაღრმავება, depth ეფექტი.
Range-ის სწორი არჩევა — ინსტრუმენტის გამოყენების ოსტატობა.

🛡

არადესტრუქციული გამოყენება (Non-Destructive Workflow)

Dodge, Burn და Sponge პირდაპირ სურათს ცვლიან — undo-ს გარდა, ცვლილება მუდმივია. პრო-გზა: შექმენი ახალი ფენა → Fill 50% Gray → Blend Mode: Overlay (ან Soft Light). ამ ნეიტრალურ ფენაზე Dodge/Burn გამოიყენე — ორიგინალი ხელუხლებელი დარჩება, ეფექტი — ცალკე ფენაზე.

  • Dodge ლოკალური გაანათება ☀️
  • Burn ლოკალური დამუქება 🔥

🎯 პროფესიული კონტროლი
