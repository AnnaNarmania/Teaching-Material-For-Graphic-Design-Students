01 · ინსტრუმენტები

Blur · Sharpen · Smudge

სამივე ინსტრუმენტი Toolbar-ში ერთ ჯგუფშია (R). ისინი პიქსელების მნიშვნელობებს — ფერს, კონტრასტს, პოზიციას — უშუალოდ ცვლიან.

💧
Blur Tool
გლუვება · ბუნდოვანება
ახლო პიქსელების ფერებს საშუალო მნიშვნელობამდე ახლოებს. კიდეები ქრება, გადასვლები რბილდება. ხატვისას — კანი გლუვდება, ფონი ლბება, კიდეები — თავისუფლდება.
R → Blur Tool
💎
Sharpen Tool
გამახვილება · კონტრასტი
ახლო პიქსელებს შორის კონტრასტს ზრდის — კიდეები მკვეთრი ხდება, ტექსტურა — გამოკვეთილი. ოდნავ გამოიყენება: გადამეტება ნოიზს და არტეფაქტებს ქმნის.
R → Sharpen Tool
🌀
Smudge Tool
გასვრა · მოძრაობა
პიქსელების ფერს ათრევს — ისე, როგორც სველ საღებავს თითით გაატარებ. Finger Painting — ფუნჯის ფერით იწყება მოძრაობა. ხელოვნურ, ილუსტრაციულ ეფექტებში.
R → Smudge Tool
💧

Blur Tool — პიქსელების "გარეცხვა"

Blur Tool ფუნჯის ქვეშ მოხვედრილ პიქსელებს Gaussian Blur-ის ლოგიკით ამუშავებს — ახლო პიქსელების ფერების საშუალო გამოაქვს.

Strength: რამდენად ძლიერია ეფექტი ერთ გავლაში (1–100%). დაბალი Strength + მრავალი გავლა = უფრო ნატიფი კონტროლი.
Mode: Normal (ყველა არხი), Darken (მხოლოდ ბნელი), Lighten (მხოლოდ ნათელი).
Sample All Layers: ყველა ხილული ფენიდან ნიმუშის აღება.

პრაქტიკა: კანის ნაოჭები, ფონის ლბება, კიდის სიმახვილის მოხსნა, ჩრდილის გლუვება.
R → Blur Toolbar → R (გრძელი დაჭ.)
💎

Sharpen Tool — კიდის "ამოღება"

ახლო პიქსელებს შორის სხვაობას ზრდის — ბნელი უფრო ბნელდება, ნათელი — უფრო ნათელდება. ეს კიდეს მკვეთრ ხდის.

Protect Detail (Options Bar) — ჩართვისას ნოიზის შემცირება და დეტალების შენარჩუნება. გირჩევ ყოველთვის ჩართო.
Strength: 20–40% — ოპტიმალური; 80%+ — ნოიზი და "ჰალო ეფექტი" კიდეებზე.
Sample All Layers: ყველა ფენიდან კითხვა.

⚠ გადამეტება: Sharpen Tool-ის ბოროტად გამოყენება ქმნის "ჰალო" (თეთრ კიდეს), ნოიზს და პლასტიკურ სახეს.
R → Sharpen Toolbar → R (გრძელი დაჭ.)
🌀

Smudge Tool — "სველი საღებავი"

Smudge Tool სიმულაციას უკეთებს თითით საღებავის გაყვანას — მოძრაობის მიმართულებით პიქსელების ფერი "ათრევს".

Strength: 50% = ზომიერი ათრევა; 100% = ძლიერი, ტალღური.
Finger Painting (Options Bar): ჩართვისას — ფუნჯის წინა-პლანის ფერით იწყებს სვლას, შემდეგ ახლო ფერებს ათრევს. გამორთვა — ახლო პიქსელებს ათრევს პირდაპირ.
Sample All Layers: ყველა ფენიდან კითხვა.

პრაქტიკა: ბეწვის, ჯვარედინი კიდეების, ილუსტრაციის ეფექტი. ასევე — კბენები, სინათლის ეფექტი.
R → Smudge Toolbar → R (გრძელი დაჭ.)
პიქსელებზე ეფექტი — ვიზუალური მაგალითი
💧 Blur — გლუვება
💎 Sharpen — გამახვილება
🌀 Smudge — ათრევა

03 · შედარება

ინსტრუმენტების კლასიფიკაცია

სად, როდის, რომელი?

Blur Tool · გამოყენება

როდის ვიყენებ?

ზედაპირის გლუვება, ფონის ლბება, კიდის მოხსნა.

  • კანი ნაოჭი, ლაქა, ტექსტ. გლ.
  • ფონი სიღრმის ეფექტი (Bokeh)
  • კიდე მკვეთრი კონტ. მოხსნა
  • ჩრდილი Drop Shadow-ს გლ.
Sharpen Tool · გამოყენება

როდის ვიყენებ?

ტექსტ. გამოკვეთა, კიდის მახვილება — ზომიერად.

  • თვალი ირისსა და წამწამი
  • ბეწვი ცხოველის ბეწვის ტექსტ.
  • ქვა მინერალის, ქვის სტრუქტ.
  • ⚠ ფრთხილად კანზე არასოდეს!
Smudge Tool · გამოყენება

როდის ვიყენებ?

ათრევა, ფერის გადადინება, ილუსტრაციული ეფექტი.

  • ბეწვი ბეწვის სტრანდები
  • ილუსტ. ციფრული ფერწერა
  • კიდე ფერის "გაყვანა"
  • ეფ. ქარი, სისწრაფე, დინება
Filter vs Tool

Filter Blur vs Blur Tool

ორი განსხვავებული მიდგომა — ერთი და იმავე ეფექტისთვის.

  • Filter Blur მთელ ფენაზე, ზუსტი
  • Blur Tool ხელით, ლოკალური
  • Filter Sharp. Smart Sharpen — უკეთ.
  • Sharp. Tool სელექტიური, ზომიერი
სრული შედარება

⚡ პიქსელებზე ზემოქმედება

  • Blur — საშუალო → ფერები ახლოვდება
  • Sharpen — სხვაობა → ფერები შორდება
  • Smudge — ათრევა → ფერი მოძრაობს

🎯 Protect Detail — Sharpen-ის

  • ჩართ.: ნოიზი მცირდება
  • გამ.: სწრაფი, ნოიზიანი
  • ყოვ. ჩართული = გირჩევ!

🌀 Finger Painting — Smudge-ის

  • ჩართ.: FG ფერით იწყება
  • გამ.: ახლო პიქ. ათრევა
  • ფერების "შერევის" ეფექტი

🖌 ოსტატის წესი

  • Blur — ნელა, დაბ. Strength
  • Sharpen — ზომ. + Protect
  • Smudge — მოკ. სვლები

🔑 სამი ინსტრუმენტი — ერთი ჯგუფი (R)

Blur, Sharpen და Smudge ერთ კლავიშზე — R — სხდება. Shift+R — გადართვა. ფუნჯის ზომა: [ / ] კლავიშებით. Hardness: Shift+[ / Shift+]. Opacity/Strength: ციფრული კლავიშები (1=10%, 5=50%).

ყველა სამი — დესტრუქციული (Original ფენა იცვლება). გამოიყენე Duplicate Layer (Ctrl+J) — ყოველთვის ასლზე.
