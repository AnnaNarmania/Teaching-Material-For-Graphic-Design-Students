01 · ფერთა მოდელები

Color Modes

ფერთა მოდელი განსაზღვრავს როგორ ინახავს და წარმოქმნის პროგრამა ფერს. შეიცვლება: Image → Mode

◼
Bitmap
მხოლოდ შავი და თეთრი — შუალედური ტონები არ არსებობს.
Mono
პიქსელი ან 0 (შავი) ან 1 (თეთრი). გამოიყენება ლოგოებში, ტექნიკურ ნახაზებში, კალიგრაფიაში. ჯერ Grayscale-ზე გადასვლა საჭიროა.
Image → Mode → Bitmap
▓
Grayscale
256 ნაცრისფერი ტონი — სინათლის ინტენსივობა 0-დან 255-მდე.
Mono
ფოტო ინარჩუნებს სინათლისა და ჩრდილის დეტალებს. გამოიყენება შავ-თეთრ ფოტოგრაფიაში, ბეჭდვაში.
Image → Mode → Grayscale
🔴
RGB Color
Red · Green · Blue — სინათლის შერევის პრინციპი. ეკრანის სტანდარტი.
Screen
R+G=Yellow · R+B=Magenta · G+B=Cyan · სამივე=White. ვებ, სოციალური მედია, მობილური, ტელევიზია. Photoshop-ის ნაგულისხმევი რეჟიმი.
Image → Mode → RGB Color
🖨
CMYK
Cyan · Magenta · Yellow · Black — მელნის შერევა. ბეჭდვის სტანდარტი.
Print
RGB-ზე ფერთა დიაპაზონი მცირეა. RGB→CMYK კონვერტაციისას ფერები ხშირად ფერმკრთალდება. ჟურნალები, პლაკატები, ბროშურები.
Image → Mode → CMYK Color
◎
Lab Color
L (სინათლე) · a (მწვანე↔წითელი) · b (ლურჯი↔ყვითელი)
Pro
მოიცავს ადამიანის თვალისთვის ხილულ თითქმის ყველა ფერს. გამოიყენება პროფესიულ ფერის კორექციასა და რესტავრაციაში.
Image → Mode → Lab Color

▸ დააჭირე ბარათს დეტალების სანახავად

02 · ტონალური კორექცია

Image Adjustments

ტონალური კორექცია — სინათლის, კონტრასტისა და ფერის დარეგულირება. გზა: Image → Adjustments

☀️

Brightness / Contrast

Brightness — მთლიანი ნათელობა. Contrast — სინათლისა და ჩრდილის განსხვავება. Contrast-ის გაზრდა ნათელს კიდევ უფრო ანათებს, მუქს კი — ამუქებს.

📷

Exposure

Exposure — მთლიანი სინათლე · Offset — ტონების გადანაწილება · Gamma — შუა ტონები. ჰგავს კამერის ექსპოზიციის კონტროლს.

vs Brightness: Brightness მარტივად ცვლის ნათელობას; Exposure კამერის ლოგიკით მუშაობს.
🎨

Vibrance

ზრდის ფერების ინტენსივობას, მაგრამ უფრო ძლიერად მოქმედებს სუსტ ფერებზე. იცავს კანის ბუნებრივ ტონებს. პორტრეტები, ლანდშაფტები.

vs Saturation: Saturation ყველა ფერს ერთნაირად ცვლის; Vibrance — სელექტიურად.
🌈

Hue / Saturation

Hue — ცვლის ფერის ტიპს · Saturation — ფერის ინტენსივობა · Lightness — ტონის ნათელობა. შეიძლება კონკრეტულ ფერებზეც გამოიყენო.

⚖️

Color Balance

სამი ღერძი: Cyan↔Red · Magenta↔Green · Yellow↔Blue. გამოიყენება თეთრი ბალანსის კორექციისა და თბილი/ცივი ტონის შესაქმნელად.

vs Hue: Hue ცვლის ფერის ტიპს; Color Balance — ფერებს შორის ბალანსს.
◑

Black & White

ფერადი ფოტო გადადის შავ-თეთრში, მაგრამ სხვადასხვა ფერის ტონები ცალ-ცალკე კონტროლდება. წითელი შეიძლება მუქი გახდეს, ყვითელი — ნათელი.

vs Grayscale: Grayscale მთლიანად შლის ფერს; B&W ტონალობის კონტროლს გვაძლევს.
🎞

Photo Filter

ამატებს ფოტოგრაფიული ლინზის ფილტრის ეფექტს. Warming Filter — თბილი ტონი · Cooling Filter — ცივი ტონი. ატმოსფეროს შეცვლა, კინემატოგრაფიული ეფექტი.

03 · შედარება

ყველაზე ხშირი გაუგებრობა

ამ წყვილებს შორის განსხვავების გაგება ზოგჯერ ყველაზე რთულია.

Vibrance

  • სუსტ ფერებზე მოქმედებს ძლიერად
  • კანის ტონი დაცულია
  • უფრო ბუნებრივი შედეგი
  • პორტრეტებისთვის იდეალური

Saturation

  • ყველა ფერს ერთნაირად ცვლის
  • კანის ტონი არ არის დაცული
  • ადვილად ხდება ხელოვნური
  • მარტივი ეფექტებისთვის

Exposure

  • კამერის ექსპოზიციის სიმულაცია
  • Gamma — შუა ტონების კონტროლი
  • Offset — ტონების გადანაწილება
  • პროფესიული ფოტო-კორექცია

Brightness

  • მარტივი ნათელობის ცვლილება
  • მთლიან სურათზე ერთნაირად მოქმედებს
  • შუა ტონების კონტროლი არ არის
  • სწრაფი, მარტივი კორექცია

Black & White

  • ფერის ტონებს ინარჩუნებს
  • ყოველ ფერს ცალ-ცალკე მართავ
  • პროფესიული B&W კონვერტაცია

Grayscale (Mode)

  • ფერის ინფორმაციას მთლიანად შლის
  • კანალები 3-დან 1-ზე მცირდება
  • ტონების კონტროლი შეუძლებელია
