02 · შედარება

რომელი როდის?

ხუთი ინსტრუმენტი გარეგნულად მსგავსია — მათ შორის განსხვავება კი ყველაზე მნიშვნელოვანია.

ინსტრუმენტი პრინციპი წყარო საუკეთესო გამოყენება
Clone Stamp
S 		ზუსტი კოპია — ფერი + ტექსტურა პირდაპირ გადადის Alt + Click (სავალდებულო) მკაფიო ტექსტურების კოპირება, ფოტომანიპულაცია
Pattern Stamp
S→Shift 		Pattern (ნიმუში) დაიხატება — არა ფოტოდან კოპია Pattern Library / Define Pattern განმეორებადი ნიმუშები, ფონის შევსება
Healing Brush
J 		ტექსტურა კოპირდება + ფერი ადაპტირდება გარემოსთან Alt + Click (სავალდებულო) კანის ნაოჭები, ხაზები — ბუნებრივი შედეგი
Spot Healing
J→Shift 		ავტომატური — Photoshop ირჩევს წყაროს Content-Aware AI-ით არ სჭირდება პატარა ლაქები, მძიმეები — ერთი კლიკი
Patch Tool
J→Shift 		Selection + Drag — დიდი ფართობის შეცვლა Drag (Source/Destination) დიდი ობიექტების წაშლა, ფართო ფონის გამოსწორება
Clone Stamp vs Healing Brush

Clone Stamp — ზუსტი კოპია. თუ ფონი განსხვავდება, "ნაჭდევი" ჩანს.

Healing Brush — ტექსტურა კოპირდება, ფერი კი ადაპტირდება. კანის რეტუშში უმჯობესი.

Spot Healing vs Patch Tool

Spot Healing — სწრაფი, ავტომატური, პატარა ფართობებისთვის.

Patch Tool — ნელი მაგრამ კონტროლირებადი. დიდი ობიექტების, ფართო ხარვეზების გამოსწორება.

03 · სამუშაო პროცესი

პრაქტიკული სამუშაო

კანის რეტუშის სტანდარტული მიდგომა — დესტრუქციული ეფექტების გარეშე.

01

ახალი Layer-ის შექმნა

ყოველთვის ახალ, ცარიელ Layer-ზე მუშაობა ორიგინალს იცავს. Ctrl+Shift+N → ახალი Layer.

02

Spot Healing Brush — სწრაფი ეტაპი

პირველ ეტაპზე Spot Healing-ით (J) სწრაფად გაასუფთავე ყველა პატარა ლაქა, მძიმე, მტვერი. Sample: All Layers.

03

Healing Brush — ნაოჭები და ხაზები

Healing Brush-ით (J, Alt+Click წყარო) დაუმუშავე სახის ხაზები. Sample: Current & Below. Opacity შეამცირე 70–80%-მდე ბუნებრიობისთვის.

04

Patch Tool — დიდი ფართობები

Patch Tool-ით (Shift+J) გამოყე დიდი ხარვეზი, გადაათრიე სუფთა ფართობზე. Content-Aware რეჟიმი ყველაზე კარგ შედეგს იძლევა.

05

Clone Stamp — ზუსტი დეტალები

სადაც ადაპტაცია არ გვჭირდება (ბეწვი, ქსოვილი, ჭადრაკი) — Clone Stamp (S) ზუსტ კოპიას იძლევა. Aligned: OFF = ყოველ გავლებაში წყარო იმავე ადგილიდან.

06

Opacity შემცირება + შედეგის შემოწმება

Layer-ის Opacity 70–85% — ოდნავ ორიგინალის ტექსტურა ჩანს. Before/After: Layer-ის თვალის ხატი ჩართე/გამორთე.
