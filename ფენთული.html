01 · შესავალი

რა არის Pen Tool?

Pen Tool Photoshop-ის ყველაზე ძლიერი ვექტორული ინსტრუმენტია. იგი ქმნის მათემატიკურ მრუდებს — Bézier curves — რომლებიც ზუსტი, მასშტაბირებადი კონტურებია.

02 · Anchor Points

Anchor Points — ტიპები

Path შედგება Anchor Point-ებისგან. ყოველ წერტილს შეიძლება ჰქონდეს 0, 1 ან 2 სახელური (Direction Handles), რომლებიც მრუდს აკონტროლებს.

Corner Point

კუთხოვანი წერტილი — მკვეთრი კიდე. Direction Handle არ ამოდის. უბრალოდ კლიკი.

მკვეთრი მრუდი

Smooth Point

გლუვი წერტილი — ორი Direction Handle ერთ ხაზზეა. კლიკი + გადათრევა.

გლუვი მრუდი

Cusp Point

კომბინირებული — ორი Direction Handle სხვადასხვა მიმართულებით. Alt + კლიკი სახელურზე.

შერეული

ნახევრად-გლუვი

ერთი Direction Handle — ერთი მხარე გლუვია, მეორე სწორი. Convert Point Tool-ით შექმნა.

ასიმეტრული

Direction Handle: ყოველ Anchor Point-ს შეიძლება ჰქონდეს 2 სახელური. სახელურის სიგრძე მრუდის სიმრგვალეს განსაზღვრავს, ხოლო მიმართულება — კუთხეს.

03 · შექმნის მეთოდები

Path-ის შექმნა

Pen Tool-ით Path-ის შექმნა სამი ძირითადი მოქმედებისგან შედგება. ზუსტი ტექნიკა განსაზღვრავს კონტურის ხარისხს.

1

სწორი ხაზების გავლება — Corner Points

უბრალოდ დააკლიკეთ (გადათრევის გარეშე) სხვადასხვა ადგილას. Photoshop ყოველ ორ წერტილს პირდაპირი ხაზით შეაერთებს. გეომეტრიული ფორმების — სამკუთხედი, მრავალკუთხა — სწრაფი შექმნისთვის.

2

მრუდი ხაზების გავლება — Smooth Points

კლიკი + გადათრევა — Anchor Point გამოჩნდება და Direction Handle-ები გამოვა. სახელურის სიგრძე მრუდის სიმრგვალეს, მიმართულება — კუთხეს განსაზღვრავს. S-ფორმის, წრის და ელიფსის შექმნისთვის.

3

Path-ის დახურვა

პირველ (საწყის) Anchor Point-ზე კლიკი — Pen Tool-ის გვერდით გამოჩნდება პატარა ○ ნიშანი. ეს ნიშნავს, რომ Path დაიხურება. ღია Path-ის დასასრულებლად — Enter ან სხვა ინსტრუმენტზე გადასვლა.

4

Corner → Smooth გადასვლა ერთ Path-ში

მრუდი ხაზის შემდეგ კუთხოვანი გაგრძელება: Alt + კლიკი ბოლო Anchor Point-ზე — ერთი Direction Handle "გაქრება" და შემდეგი სეგმენტი პირდაპირი ხაზი გახდება. ეს ტექნიკა ამოჭრისას ხშირად გამოიყენება.

5

Freeform Pen Tool

Pen Tool-ის ალტერნატიული ვარიანტი — თავისუფლად ხატვა თითის ან მაუსის გადათრევით. Anchor Point-ები ავტომატურად ემატება. ნაკლებად ზუსტია, მაგრამ სწრაფია ორგანული ფორმებისთვის. Options Bar-ში Magnetic ჩართვა მკიდეს "ეწებება".

04 · რედაქტირება

კონტურის რედაქტირება

Path-ის შექმნის შემდეგ ნებისმიერ დროს შეიძლება ზუსტი კორექტირება. სწორი ინსტრუმენტის არჩევა — გასაღები.

Direct Selection Tool

A — Anchor Point-ის მართვა

ცალკეული Anchor Point-ების და Direction Handle-ების გადაადგილება. Path ნაწილობრივ ლოკალურად იცვლება.

  • კლიკი წერტილზეერთი Anchor Point შეირჩევა
  • Shift + კლიკირამდენიმე წერტილი ერთად
  • კლიკი + გადათრევაწერტილის გადაადგილება
  • Handle-ის გადათრევამრუდის ფორმის შეცვლა
  • Deleteშერჩეული წერტილის წაშლა
Path Selection Tool

Shift+A — მთელი Path-ის მართვა

Path-ის მთლიანად გადაადგილება — ყველა წერტილი ერთად მოძრაობს. რამდენიმე Sub-path-ის ერთდროული მართვა.

  • კლიკი Path-ზემთელი Path შეირჩევა
  • გადათრევამთელი Path გადაადგილდება
  • Ctrl+Aყველა Path შეირჩევა
  • Alt + გადათრევაPath-ის კოპირება
Add / Delete Anchor Point

+ / − ინსტრუმენტები

Path-ზე კლიკი Add-ით — ახალი წერტილი. არსებულ წერტილზე Delete-ით — ამოღება Path-ის გაწყვეტის გარეშე.

  • კლიკი სეგმენტზეახალი Anchor Point ემატება
  • კლიკი წერტილზეAnchor Point იშლება
  • ავტომატურიPen Tool სეგმენტზე → Add, Point-ზე → Delete
Convert Point Tool

Corner ↔ Smooth გადაქცევა

Smooth Point-ს კუთხოვანად ან კუთხოვანს გლუვად გადაქცევა. Pen Tool-ის გამოყენებისას Alt ხდება Convert Point.

  • კლიკი Smooth-ზეCorner-ად იქცევა (Handle ქრება)
  • კლიკ+გადათრევა Corner-ზეSmooth-ად იქცევა
  • Alt + Handleერთი Handle დამოუკიდებელი ხდება

Path → Selection გადაქცევა

Path-ის შექმნის შემდეგ: Ctrl + Enter (⌘ + Return) — Path ხდება Selection (მოციმციმე ხაზი). ან Paths Panel-ში: "Load path as selection" ღილაკი. Feather Radius Options-ში განსაზღვრავს კიდის სიმხოლოვეს.

⚠ Path ინახება Paths Panel-ში — Work Path სახელით. შენახვისთვის დააორჯერეთ კლიკი და დაარქვით სახელი, წინააღმდეგ შემთხვევაში ახალი Path გადაწერს.
